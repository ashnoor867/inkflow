# InkFlow

This is a blog app where you can write your thoughts.

## What it does

- Post blogs
- Login with GitHub
- Read other people's blogs

## How to run

1. Clone this project
2. Run `npm install`
3. Create a `.env` file with:
   - MONGODB_URI
   - SESSION_SECRET
   - GITHUB_CLIENT_ID
   - GITHUB_CLIENT_SECRET

4. Start the app:
   ```
   npm start
   ```

Or use dev mode:
```
npm run dev
```

## Made with

- Express
- MongoDB
- Handlebars
- Passport for GitHub login
